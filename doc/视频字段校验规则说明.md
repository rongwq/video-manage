# 视频新增/修改功能 - 字段校验规则说明

## 一、核心目标

为视频管理系统的「视频新增」「视频修改」接口实现前后端一致性校验，明确每个字段的校验维度、错误提示、场景差异（新增/修改）。

## 二、基础约定

1. **校验执行时机**：
   - 前端：提交表单时（即时校验）
   - 后端：接口接收参数时（最终校验）

2. **错误反馈**：所有校验失败需返回「字段名 + 具体错误原因」（如：title 不能为空）

3. **场景区分**：明确字段在「新增」「修改」场景下的校验差异（如：id 仅在修改时校验）

## 三、实体字段校验规则

### 1. 字段名：title（类型：String）

| 校验项 | 规则 | 新增场景 | 修改场景 |
|--------|------|----------|----------|
| 非空 | 不能为空 | ✅ | ✅ |
| 长度 | 1~100 字符 | ✅ | ✅ |
| 字符范围 | 仅允许中文、英文、数字、-_ | ✅ | ✅ |

**错误提示示例**：
- 视频标题不能为空
- 标题长度需在 1-100 字符之间
- 标题包含非法特殊字符，仅允许中文、英文、数字、-_

**前后端校验要求**：前后端均校验非空 + 长度 + 字符范围

---

### 2. 字段名：money（类型：Integer）

| 校验项 | 规则 | 新增场景 | 修改场景 |
|--------|------|----------|----------|
| 收费视频(type=2)时 | 必填且数值 > 0 | ✅ | ✅ |
| 非收费视频(type≠2)时 | 必须为 null/0 | ✅ | ✅ |
| 数值上限 | ≤9999 | ✅ | ✅ |

**错误提示示例**：
- 收费视频时金额不能为空
- 金额必须大于 0
- 金额不能超过 9999
- 非收费视频时金额需为 0 或空

**前后端校验要求**：前后端均校验 type 关联规则 + 数值范围

---

### 3. 字段名：url（类型：String）

| 校验项 | 规则 | 新增场景 | 修改场景 |
|--------|------|----------|----------|
| 非空 | 不能为空 | ✅ | ✅ |
| URL格式 | http/https 开头 | ✅ | ✅ |
| 视频格式 | mp4/avi/mov/rmvb/flv/wmv/mkv | ✅ | ✅ |
| 长度 | ≤500 字符 | ✅ | ✅ |

**错误提示示例**：
- 视频 URL 不能为空
- 视频 URL 格式错误，必须以 http:// 或 https:// 开头
- 仅支持 mp4、avi、mov、rmvb、flv、wmv、mkv 格式的视频
- 视频 URL 长度不能超过 500 字符

**前后端校验要求**：前后端均校验非空 + URL 格式 + 长度 + 视频格式

---

## 四、补充校验规则（全局/逻辑校验）

### 1. 通用格式校验

- **字符串字段**：所有字符串字段先去除首尾空格再执行校验（如 title、url）
- **数值字段**：禁止输入非数字字符（如 money、likeNum、playNum）

### 2. 逻辑关联校验

| 视频类型 | type值 | 关联规则 |
|----------|--------|----------|
| 免费无广告 | 1 | adList 必须为空，且 money 必须为 0 |
| 收费视频 | 2 | money 必须 > 0 |
| 免费有广告 | 3 | adList 必须非空，且 money 必须为 0 |

**错误提示示例**：
- 免费无广告视频不能配置广告列表
- 免费无广告视频金额必须为 0
- 免费含广告视频必须配置广告列表
- 免费含广告视频金额必须为 0

### 3. 后端额外校验

- **标题唯一性**：视频标题 title 在系统内唯一
  - 新增时：全量校验
  - 修改时：排除自身 ID 后校验

**错误提示示例**：
- 视频标题已存在，请更换标题
- 视频标题已被其他视频使用，请更换标题

## 五、代码实现说明

### 后端实现

**文件位置**：`ruoyi-common/src/main/java/com/ruoyi/common/utils/VideoValidationUtils.java`

**核心方法**：
- `validateForAdd(Video video)` - 新增场景校验
- `validateForUpdate(Video video)` - 修改场景校验
- `validate(Video video, ValidationScene scene)` - 通用校验
- `validateAndThrow(Video video, ValidationScene scene)` - 校验并抛出异常

**Service层集成**：`VideoServiceImpl.java`
- `insertVideo()` - 新增时调用校验
- `updateVideo()` - 修改时调用校验

### 前端实现

**文件位置**：`ruoyi-ui/src/utils/videoValidation.js`

**核心方法**：
- `validateTitle(title)` - 标题校验
- `validateUrl(url)` - URL校验
- `validateMoney(money, type)` - 金额校验
- `validateVideo(video, scene)` - 完整校验
- `generateFormRulesWithForm(getForm)` - 生成 Element UI 表单规则

**页面集成**：`views/video/video/index.vue`
- 导入校验工具
- 表单 rules 使用生成的校验规则
- submitForm 中执行完整校验
